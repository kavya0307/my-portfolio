# AWS S3 + CloudFront Portfolio Website

## Objective
Host a static portfolio website using AWS S3 and CloudFront.

## Technologies Used
- HTML
- CSS
- Git
- GitHub
- AWS S3
- AWS CloudFront

## Features
- Responsive Portfolio
- Hero Section
- About Section
- Project Cards
- Contact Section
- HTTPS Delivery via CloudFront

## Deployment Steps
1. Created portfolio website using HTML/CSS.
2. Uploaded source code to GitHub.
3. Hosted website on AWS S3.
4. Configured CloudFront for HTTPS delivery.
